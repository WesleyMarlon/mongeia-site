Como começar?

Tudo começa pelo HTML, ele é o esqueleto do seu site.

Isso não é uma simples frase que os programadores ficam repetindo. O HTML é o conteúdo do seu site, esse texto que você está lendo são HTML.

Entretanto, HTML não é somente textos, como eu disse, ele é o conteúdo, tudo que está no site é HTML. Mas nem tudo é HTML, ele é o conteúdo e o CSS é o que você vê. O HTML não precisa do CSS para mostrar seus conteúdos. No entanto, se você quiser que seu site não pareça um site pré anos 2000, você vai precisar do CSS.
