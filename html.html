HTML: A estrutura do site

Eu quis começar ensinando sobre Frontend. Por quê? É o mais fácil de aprender e o que mais rápido te coloca no mercado. Se você começar pelo Frontend, sua mente abre para o mercado. Porque com a trindade do Frontend: HTML, CSS e JS, você consegue fazer o site da padaria que você vai todo dia. Simples assim. Você deixa de ser um programador e começa a pensar no mercado. Começa aprendendo a vender o seu serviço.

O que é HTML?

Tudo começa pelo HTML, ele é o esqueleto do seu site.

HTML é o conteúdo do seu site. Esse texto que você está lendo é HTML.

HTML é todo conteúdo: texto, imagem, vídeo, botão. CSS é a aparência. HTML funciona sem CSS, mas fica com cara de site anos 2000. Navegador usa um estilo básico se você não escrever CSS.

Quer um site simples? Use a fórmula: HTML + CSS = Frontend

Quer ir além e não ser só "revista"? Quer um aplicativo em formato de site? Você vai precisar de JavaScript. Não tem como fugir. Mas não corra. Abrace a dificuldade. Se é difícil, poucos fazem. Se poucos fazem e tem demanda, o dinheiro vai pra quem sabe.

Como começar?

Baixe o VS Code.

Não vou te enrolar. Em 2026, mais de 75% dos devs usam VS Code. Recomendar outro seria loucura.

Vou te dar o caminho. Os detalhes você pesquisa. Mas não te deixo no escuro:

  • Aprenda a instalar o VS Code. Pode ser vídeo do YouTube. Recomendo.
  • Nome de pasta é sagrado: Nunca use "ç", acento ou espaço. Use meu-projeto ou meu_projeto. Evita bug que trava tudo.
  • Em resumo: Instale o VS Code e crie pastas do jeito certo.
