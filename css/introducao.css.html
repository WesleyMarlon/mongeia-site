CSS: A aparência do site

A área de navegação de fato é a área da programação com o início mais fácil de todos. Talvez perca somente para o Python em relação a early game fácil. Se você escolheu essa área e ainda não tem pelo menos a Introdução do HTML. Sim, de fato estou te expulsando. Não por mal. Mas se você não tiver a base do HTML, tudo que for aprender não servirá de nada. Tentar aprender CSS primeiro que HTML é igual querer pintar uma casa sem ter paredes. Entende agora porque não funciona?

O que é CSS?

Enquanto o HTML são a estrutura da casa. O CSS é a parte estética.

Sem o CSS esse site seria feio. Mas não é "feinho". Ele seria HORROROSO. Quer que eu prove? Pois bem: Esse mesmo site, mas sem CSS. Se clicou, você viu o quão feio é sem CSS.

CSS é toda a aparência: cor, tamanho, posição, fonte, espaçamento. HTML é o esqueleto. CSS é a roupa, o corte de cabelo e a postura.

Não se esqueça. HTML funciona sem CSS, mas CSS não funciona sem HTML. Porque o CSS precisa de um <button> pra pintar, de um <div> pra centralizar. Sem HTML, o CSS fica sem emprego.
