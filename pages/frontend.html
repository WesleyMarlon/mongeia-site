Frontend: É TUDO que o usuário vê e toca

Pense assim: se tem pixel na tela e o usuário interage, é frontend. A esmagadora maioria das coisas que você vê são feitas com HTML + CSS = Frontend. Entretanto, existe o Backend, que é a parte do servidor. Ele pode ser feito em JavaScript com Node.js, Python, Java, PHP, etc. Um site pode ser feito 100% em Frontend, mas a maioria tem Backend pra salvar dados, login, etc. O YouTube sem backend vira só uma página bonita que não carrega vídeo nenhum.

Usos do Frontend

Frontend mexe com:

  • Sites/Web: Instagram Web, YouTube
  • Apps Mobile: Instagram App, Nubank, iFood
  • Desktop: VSCode, Discord, Spotify Desktop
  • Smart TV: Netflix na TV, YouTube na TV
  • Jogos: Menu, HUD, Inventário do GTA
  • Carro/Relógio: Painel da Tesla, multimídia, interface

Tipos de Frontend

A base do Frontend Web é a trindade: HTML + CSS + JS
Mas "Frontend" é qualquer interface que o usuário toca. Por isso existem vários tipos:

  • Navegador: HTML, CSS, JS, React, Vue, Svelte
  • Celular App: React Native, Flutter, Swift, Kotlin
  • PC/Mac App: Electron, Tauri, WPF, Qt
  • Jogos: Unity UI, Unreal UMG, Scaleform
