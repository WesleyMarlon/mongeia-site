HTML: A estrutura do site

Esse é o nível intermediário do HTML. Anteriormente foi ensinado o básico dessa estrutura. No entanto, um site bom deve ser organizado. Eu recomendo que antes de dar inicio a Introdução do CSS, você deve ter uma boa base do HTML, pois caso o contrário isso vai prejudicar o seu progresso no CSS. Um site desorganizado só serve para dar dor de cabeça. Uma página em média desse site tem 300 linhas de código. Agora imagine um site com várias páginas com cada uma tendo 300 linhas. Se não organizar, isso vira uma bagunça e por não dedicar tempo organizando o código, você vai acabar gastando mais tempo ainda para configurar o site.
